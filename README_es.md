# Proyecto de Acortador de URLs

Este proyecto es una aplicación para acortar URLs, gestionar su ciclo de vida y redirigir a los usuarios a las URLs originales. Incluye funcionalidades como la creación de URLs cortas, redirección, manejo de expiración y límites de uso.

## Características
- Crear URLs cortas a partir de URLs originales.
- Redirigir a la URL original utilizando el código corto.
- Manejo de expiración de URLs.
- Límite de accesos configurables para cada URL corta.
- Persistencia de datos mediante un repositorio.

## Requisitos Previos
- .NET 9 instalado en su sistema.
- Un editor de código como Visual Studio o Visual Studio Code.
